Glossaries

商品を判別可能 判別不能
認識結果が一致 TP FP (常に0)
認識結果が不一致 FN FN
認識不能 FN TN

正解率(accuracy) = (TP + TN) / (TP + FP + FN + TN)

不正解率(error) = (FP + FN) / (TP + FP + FN + TN) = 1 - 正解率

適合率(presicion) = TP / (TP + FP) (常に1)

再現率(recall) = TP / (TP + FN)

F値(f-measure) = 2 x 適合率 x 再現率 / (適合率 + 再現率)

これらは本来、不良品検査の分野で使われている指標です（少々無理にこじつけています）。 正解率と再現率に注目して精度を高めることを目指します。
